# Sales Data Analysis Dashboard

##  Overview

This project analyzes retail sales data to identify customer purchasing behavior, product performance, and regional sales trends using SQL and Power BI.

---

## Business Problem

The business wanted to understand:

- Which products generate the highest revenue?
- Which regions perform best?
- Which customer segments contribute the most sales?
- What are the monthly sales trends?

---

Retail Sales Dataset

- Revenue
- Profit

## Key Insights

- Electronics generated the highest revenue.
- South region recorded the strongest sales.
- Repeat customers contributed significantly to revenue.

---

## Business Recommendations

- Focus marketing on high-value customers.
- Expand inventory for top-selling categories.
- Improve performance in underperforming regions.
